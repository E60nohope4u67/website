Formation

Formation happends when 2 or more elements join together.

Water + Salt -> SaltWater

2H2 + O2 → 2H2O

Decomposition

Decomposition happends when 2 or more elements come apart.

Fire -> Fuel + Air

Single Replacement

Single Replacement happends when 1 element swaps with another.

Port + AirPlane -> AirPort + Plane

Double Replacement

Double Replacement happends when 2 elements swap with the another.

CopperRod + LightningBolt -> LightningRod + CopperBolt

Combustion

Combustion happends when a hydrocarbon and an oxygen are ignited to create oxygen and carbon dioxide as a by product.

Candle + Pumkin -> Jack-o-lantern + Light

CH4 + 2O2 → CO2 + 2H2O
